APPRIS Documentation
Data Production
APPRIS is a database server that deploys a range of computational methods to provide value to the annotations of the human genome. The database selects one of the CDS for each gene as the principal functional combining protein structural information, functionally important residues and evidences of non-netrual evolution of exons amongst others.
  • Gene and Transcript types: The main data set combines the HAVANA manual annotation using evidence from various sources and research groups with the Ensembl automatic annotation pipelines to achieve an accurate and complete annotation of the human genome.

  • APPRIS Pipeline: Annotations displayed on our website are products of the APPRIS automatic pipeline that are deposited into public databases.
Data Access
Report page of this web site offers an export option, suitable for small amounts of data. This is the ideal option if you want a short sequence as a FASTA file, or a GFF file of a few features in a region. Simply click on one of the "Export" links in the right menu, and select your output options. But if you wish to extract multiple features, we recommend the following alternatives.
  • Perl API Documentation: APPRIS data can be accessed remotely using our Perl API to connect to a public database server.

  • Pdoc Perl documentation: This Perl API documentation is automatically generated from POD within the modules by Pdoc.

  • Web Services: APPRIS data can be returned remotely using web services.

  • BioMart: APPRIS supports downloading of many more correlation tables via the highly customisable BioMart data mining tool.
